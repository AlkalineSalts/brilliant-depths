This contains all the work done on the game so far in Game/. Source code 
is inside src. To run this on a mac, use "open -n -a love.app Game/". To run this on windows, open the windows distribution folder and run Game.exe stored within.
If you wish to run this as a love archive, zip up the contents of Game/ and rename it to 
Game.love, download love2d at https://love2d.org/ and then open Game.love 
with it.
